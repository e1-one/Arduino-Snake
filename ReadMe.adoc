=== Folder structure

....
 snake_game                => Arduino sketch folder
  ├── snake_game.ino       => main Arduino file
  ├── pitches.h            => tune constants collection
  ├── buzzer.h             => help methods
  ├── led.h
  ├── game.h
  ├── helper.h
  └── ReadMe.adoc          => this file
....
